| Operation       | Time Complexity (Average/Amortized) | Time Complexity (Worst Case) |
| --------------- | ----------------------------------- | ---------------------------- |
| Insert          | O(log n)                            | O(log n)                     |
| Extract Min/Max | O(log n)                            | O(log n)                     |
| Peek            | O(1)                                | O(1)                         |
| Update          | O(log n)                            | O(log n)                     |
| Delete          | O(log n)                            | O(log n)                     |
| Search          | O(n)                                | O(n)                         |
| Build Heap      | O(n)                                | O(n log n)                   |
